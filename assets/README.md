# Assets

## Description:
Each asset is defined as a USD (Universal Scene Description) originating from Pixar. Some files also contain additional descriptions
and code attached through Isaac Sim's Action Graph and Omnigraph APIs to connect with ROS2 and to control internal functions.

| #   |      Asset Folder     |  Description |
|----------|:-------------:|:------:|
| 1 | camera_assets              | Holds the ECM needed to communicate and view the Surgical Robotics Challenge  |
| 2 | needle_thread_assets       | Holds different needle thread objects with different physics attached |
| 3 | patient_side_manipulator_assets               | Contains advanced and simplified models for the PSM1 and PSM 2|
| 4 | pegboard_assets     | Rigid body environment for pegboard handoffs            |
| 5 | phantom_assets            | Rigid body version of the AMBF Surgical Challenge phantom                               |
